# 📝 StickyDo
포스트잇 스타일의 Todo 관리 PWA 웹 애플리케이션

# 📌 프로젝트 소개
StickyDo는 React와 TypeScript를 기반으로 제작한  
포스트잇 스타일의 Todo 관리 PWA 웹 애플리케이션입니다.

웹과 모바일 환경에서 설치형 앱처럼 사용할 수 있도록 구현했으며,  
사용자의 일상 업무 관리를 돕는 것을 목표로 개발했습니다.

## ✨ 주요 기능
- 할 일 추가 / 삭제 / 완료 처리
- Redux Toolkit을 활용한 전역 상태 관리
- styled-components 기반 UI 스타일링
- PWA 지원 (홈 화면 설치 가능)

---------------------------------------------

## 📚 배운 점
- 역할 중심 컴포넌트 설계를 통해 재사용성을 고려한 UI 구조를 설계하는 경험을 했습니다.
- Redux Toolkit을 활용하여 전역 상태를 관리하고, 상태 흐름을 이해하게 되었습니다.
- styled-components를 활용해 전역 스타일과 컴포넌트 스타일을 체계적으로 관리하는 방법을 익혔습니다.
- PWA 환경을 구성하여 웹 애플리케이션을 설치형 앱으로 배포하는 경험을 했습니다.
- GitHub Pages를 이용한 배포 과정을 통해 실제 서비스 운영 흐름을 이해하게 되었습니다.
